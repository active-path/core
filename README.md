# ActivePath: 
ActivePath provides an interface for rendering your partials. 

## Installation 

Add to your Gemfile:

```ruby
gem 'active_path'
```

Add the initializer `config/initializers/active_path.rb` and enable `ActivePath`:

```ruby

ActivePath.configure do |config|

  config.enabled = true

end

```
